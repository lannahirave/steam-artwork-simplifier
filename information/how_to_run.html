How To Run (Short)

0) Before you run (Python + FFmpeg)

Python runs the local automation scripts:

  • video_parts_pipeline.py
  • steam_hex_patch.py
  • steam_hex_edit_header.py

FFmpeg does the actual media work (video -> GIF, resize, palette, probe).

The scripts use FFMPEG_BIN from .env (current default: D:\ffmpeg\bin).

Download links:

  • Python
  • FFmpeg official site

Quick checks:

python --version
D:\ffmpeg\bin\ffmpeg.exe -version
D:\ffmpeg\bin\ffprobe.exe -version

If any command fails, fix Python/FFmpeg path first, then run the steps below.

1) Generate GIFs from video

Workshop (5 parts, 150px each):

python video_parts_pipeline.py --input .\media\my_video.mp4 --preset workshop

Featured (single 630px):

python video_parts_pipeline.py --input .\media\my_video.mp4 --preset featured

Brief:

  • Script reads defaults from .env.
  • Precheck runs first (to catch likely too-large outputs).

If needed, bypass precheck:

python video_parts_pipeline.py --input .\media\my_video.mp4 --preset featured --skip-precheck

2) Autofill Steam upload page (DevTools Console)

  • Workshop: autofill/steam_upload_autofill_workshop.js
  • Featured: autofill/steam_upload_autofill_featured.js

Brief: sets required fields and checkbox automatically.

3) Hex tools

Patch EOF byte only:

python steam_hex_patch.py .\media\my_folder --recursive --ext .gif

Patch GIF header width/height (+ optional EOF):

python steam_hex_edit_header.py .\media\my_folder --recursive

4) Where output appears

Default output path:

.\media\<input_file_name>\output\

5) Speed / CPU usage

Use all available CPU automatically:

python video_parts_pipeline.py --input .\media\my_video.mp4 --preset workshop --max-workers 0 --ffmpeg-threads 0

Equivalent .env defaults:

GIF_MAX_WORKERS=0
FFMPEG_THREADS=0
USE_NVIDIA=false

Optional NVIDIA decode (off by default):

python video_parts_pipeline.py --input .\media\my_video.mp4 --preset workshop --use-nvidia

Oversize lossy fallback (runs only if GIF is still above max KB):

python video_parts_pipeline.py --input .\media\my_video.mp4 --preset workshop --lossy-oversize --lossy-level 2 --lossy-max-attempts 24
