Steam Artwork Toolkit - Run Guide

Primary workflow: Web app

Use the browser app in web/.

cd .\web
npm install
npm run dev

Required response headers:

  • Cross-Origin-Opener-Policy: same-origin
  • Cross-Origin-Embedder-Policy: require-corp

Legacy workflow: Python scripts

Legacy scripts remain available for compatibility.

python video_parts_pipeline.py --input .\media\my_video.mp4 --preset workshop
python video_parts_pipeline.py --input .\media\my_video.mp4 --preset featured
